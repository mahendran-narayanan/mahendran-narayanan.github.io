2. A relation ‘f’ is defined by f ( x ) = x 2 − 2 where, x ∈ { − 2 , − 1 , 0 , 3 }
(i) List the elements of f
(ii) Is f a function?
f ( x ) = x 2 – 2 where x ∈ { − 2 , − 1 , 0 , 3 }
(i)
f ( -2 ) = ( − 2 ) 2 – 2 = 2
f ( 0 ) = ( 0 ) 2 – 2 = − 2
f ( -1 ) = ( − 1 ) 2 – 2 = − 1
f ( 3 ) = ( 3 ) 2 – 2 = 7
Therefore, f = {( − 2 , 2 ),( − 1 , − 1 ) ,( 0 , − 2 ),( 3 , 7 ) }
(ii)
We note that each element in the domain of f has a unique image.
Therefore f is a function.
3. If X = {–5,1,3,4} and Y = {a,b,c}, then which of the following relations are functions from X to Y ?
(i) R1 = {(–5,a), (1,a), (3,b)}
(ii) R2 = {(–5,b), (1,b), (3,a),(4,c)}
(iii) R3 = {(–5,a), (1,a), (3,b),(4,c),(1,b)}
(i)
R1 = {(–5,a), (1,a), (3,b)}
R1 is not a function as 4 ∊ X does not have an image in Y

(ii)
R2 = {(–5,b), (1,b), (3,a),(4,c)}
R2 is a function as each element of X has an unique image in Y.

(iii)
R3 = {(–5,a), (1,a), (3,b),(4,c),(1,b)}
R3 is not a function as 1 ∊ X has two images a ∊ Y and b ∊ Y .
